# GoodBoyBingo
Content Maschine

## Desktop-App (Windows/macOS/Linux)

Diese App läuft als Desktop-Anwendung mit Electron.

### Für Nutzer (ohne Commands)

Wenn du die App nur benutzen willst, brauchst du keine Commands auszuführen:

1. Lade den Installer `GoodBoyBingo-Setup.exe` aus dem Ordner `dist/` (oder aus den Releases).
2. Öffne den Installer per Doppelklick und installiere die App.
3. Starte die App anschließend ganz normal über die Verknüpfung im Startmenü.

### Für Entwickler

```bash
npm install
npm start
```

```bash
npm run dist
```
